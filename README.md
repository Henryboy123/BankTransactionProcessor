# BankTransactionProcessor

In this project I created a NestJS server which handles incoming files and validates them,
I used fs for reading file paths, csv-parser and xml2js in order to interact with the csv files and xml files.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
